# Roadmap

## Phase 0: Bootstrap and Schemas
- Repo bootstrap and standards scaffolding.
- Initial package architecture and MCP tool skeleton.
- Versioned battery/run/report schemas.

## Phase 1: Fixed-Form Battery MVP
- Complete fixed battery execution path for real adapters.
- Harden report generation and schema validation.

## Phase 2: Stability Sweep and Comparison Reports
- Add run-to-run stability sweeps.
- Add wrapper dependence and run comparison tooling.

## Phase 3: Semi-Adaptive Stopping
- Introduce branch-based stopping rules and adaptive item selection.

## Phase 4: Rich Adapters and CI Integration
- Expand provider adapters and CI-gated eval checks.

## Phase 5: Optional Learned Scorers / Adjudicators
- Add calibrated learned scoring modules behind explicit flags.
